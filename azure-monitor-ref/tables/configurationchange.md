# ConfigurationChange

View changes to in-guest configuration data such as Files Software Registry Keys Windows Services and Linux Daemons

## Categories

- IT & Management Tools
## Solutions

- ChangeTracking
## Resource types

- Virtual machines
- VMware
- Azure Stack HCI
- System Center Virtual Machine Manager
- Virtual Machine Scale Sets
