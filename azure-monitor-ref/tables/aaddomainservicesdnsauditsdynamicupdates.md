# AADDomainServicesDNSAuditsDynamicUpdates

DNS server audit events enable change tracking on the DNS server. This table contains operational audit events for dynamic updates.

## Solutions

- LogManagement
## Resource types

- Azure AD Domain Services




## Columns

|---|---|---|
